Main routines
=============
Main routines are the heart of *spexxy*, describing a specific process that should be run on the data.

There is one pre-defined main routine in *spexxy*:

* :class:`ParamsFit <spexxy.main.ParamsFit>` takes one or more spectrum components and finds the best parameter
  combination to match a given spectrum.
